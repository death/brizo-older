Brizo history

Version 1.02

  • Brizo is now compatible with VC6.

Version 1.01

  • size_t belongs to the std namespace, and now Brizo respects that.
  • Removed old Brizo::SIB, Renamed Brizo::SibData to Brizo::SIB.
  • Doxygen documentation.
  • Opcode flags are now in an enumeration, Brizo::OpcodeFlags.
  • Renamed Brizo::MemAddr to Brizo::MemoryAddress.
  • Renamed Brizo::Direct to Brizo::DirectAddress.
  • Fixed a potential pointer out-of-bounds bug.
  • Fixed the SLDT instruction decoding.
  • Improved Brizo::Decoder and Brizo::Disassembler constructors/assignment operators efficiency.
  • SSE3 (PNI) instruction support (thanks to ParaBytes).
  • Changed behavior when decoding some invalid instructions.
  • Brizo namespace changed.
  • Source code added.
  • Minor changes.

Version 1.00

  • Initial release
